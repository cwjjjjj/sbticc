# MPI — 消费人格图鉴

> 目标：作为 GSTI → FPI → FSI 之后的低风险收官测试。MPI 不碰性别对立，不碰家庭创伤，不碰阶层审判，直接切入"我到底是怎么把钱输给世界的"这个高自嘲场景，承接 2025-2026 年"富养自己 / 抠门大师 / 二奢鉴定师"的大众话题流量。

---

## 1. 核心定位

### 1.1 命名

- **主推中文**：《消费人格图鉴》
- **备选中文**：《你是怎么输掉钱的》《钱去哪了·人格版》《破产 MBTI》
- **英文代号**：**MPI**（Money Persona Index）
- **副标题**：你是怎么把钱输给这个世界的
- **路径建议**：`/new/mpi`

### 1.2 一句话解释

31 亿剁手党里，总有一个和你一模一样的败家模式——MPI 不算你多有钱，只算你钱是怎么没的。

### 1.3 用户动机

- **想试的**：最近花得心虚，想知道自己属于哪一类"消费事故"。
- **想晒的**：结果是梗，可以配一张截图发朋友圈／小红书自嘲。
- **想转的**：把测试扔群里看谁是"直播间人质"、谁是"二奢鉴定师"，互相对号入座。
- **想截图给对象的**：配文"你看看你"，比说教好用。

### 1.4 调性分寸

**低至中度冲击，主打自嘲不主打贬低。**

- 用：剁手、回血、种草、拔草、塌房、倒爷、羊毛党、人质、永动机、狂魔
- 慎用：穷鬼、低端、破产、吃土（作为身份标签时）、底层
- 不用：任何真实品牌名、任何主播/店铺实名
- 基本守则：**笑消费行为，不笑消费能力**——花不起钱不是失败，乱花钱才是梗。

### 1.5 不做什么

- 不评估用户真实收入、负债、信用。
- 不把"低消费"写成社会边缘。
- 不做省钱鄙视消费或消费鄙视省钱的二元对立。
- 不点名任何主播、品牌、平台、APP。
- 不把"不花钱"写成贫穷或失败，也不把"花钱多"写成富裕或成功。

## 3. 维度体系

为了复用当前 `TestConfig` / matching 架构，MPI 采用 6 维，每维 `L/M/H` 三档。22 题覆盖，每维 3-4 题。

| 维度 | Code | 高分含义（A 极） | 低分含义（B 极） |
|------|------|-----------------|-----------------|
| 囤货欲 | `HOARD` | 购物车永远是满的，家里有三年份的纸巾 | 家徒四壁极简党，用完才买下一件 |
| 炫耀欲 | `FLAUNT` | 新包必须先发朋友圈，不发等于白买 | 花了钱也不说，怕别人来借钱 |
| 省钱欲 | `FRUGAL` | 算优惠券算到小数点后两位 | 贵的就是好的，便宜让我没安全感 |
| 易种草度 | `SUSCEPT` | 一条笔记就下单，第二天后悔 | 种草反而脱粉，越推越不买 |
| 二手偏好 | `SECONDHAND` | 二奢直播间蹲守者，闲鱼常驻居民 | 坚决原价新品，二手即膈应 |
| 直播沉迷度 | `LIVESTREAM` | 主播喊"321上链接"能从椅子上弹起 | 直播间不看不进不停留 |

### 3.1 两极人格速写（SBTI 辛辣风）

- **HOARD-H 囤货本能**：冰箱塞到关不上，厕所囤了 18 卷纸巾，遇到"多件优惠"就丢魂。
- **HOARD-L 极简洁癖**：家里只有床和一张椅子，一件新东西进门就像过海关。
- **FLAUNT-H 晒货磁场**：没晒的包不是包，没晒的咖啡不是咖啡。
- **FLAUNT-L 黑户消费**：买了也藏着，怕被家人骂，怕被朋友蹭。
- **FRUGAL-H 比价机器**：三家平台交叉比价 40 分钟，最后为了省 3 块钱多花 30 分钟。
- **FRUGAL-L 贵感依赖**：打折反而不敢买，"便宜没好货"是你刻在 DNA 里的。
- **SUSCEPT-H 种草即下单**：博主还没说完你已经在付款页面。
- **SUSCEPT-L 抗种草体**：别人越说好你越冷笑，推荐算法对你无效。
- **SECONDHAND-H 二手信仰**：新品是交智商税，二奢才是真爱。
- **SECONDHAND-L 新品洁癖**：别人穿过的东西想想就全身发痒。
- **LIVESTREAM-H 直播间人质**：主播一句"最后三分钟"能让你忘记自己几点睡。
- **LIVESTREAM-L 直播绝缘体**：那种 PUA 节奏对你完全没用，滑走从不停留。

### 5.4 类型描述撰写原则

每个类型描述采用四段式（沿用 SBTI / FPI 家族结构）：

1. **镜像吐槽**（1-2 句）：一针见血点出核心消费画像。
2. **天赋优势**（2-3 句）：辛辣包装的"好话"（这类人其实擅长什么）。
3. **隐秘代价**（2-3 句）：扎心点破这种消费模式的长期账单。
4. **一句刺痛你的话**（1 句）：适合截图做朋友圈配文。

**撰写红线**：

- 刺痛只刺**行为模式**，不刺**身份属性**（不写"你就是没钱"，写"你算得太细反而更累"）。
- 不点名品牌、平台、主播。
- 不嘲笑"不买"行为（`ZERO$` 要写得清醒，不写得可怜）。

## 6. 题目设计

- **总题数**：22 题
- **分布**：HOARD 4 / FLAUNT 3 / FRUGAL 4 / SUSCEPT 4 / SECONDHAND 3 / LIVESTREAM 3 + Gate 1 题（LIVESTREAM 维度兼 Gate）
- **场景覆盖**：购物车囤积 / 双11 / 直播间 / 618 / 生日礼物 / 朋友借钱 / 吃饭 AA / 搬家 / 租房 / 送礼 / 二奢直播间 / 会员续费
- **题目原则**：
  - 口语化、日常化，避免书面化问卷感。
  - 每个选项 1-3 个维度加分，不做显式标注（避免用户倒推）。
  - 不强制用户必须有消费能力——选项里永远有"我一般不会为这花钱"这档。

### 6.1 样题 4 条

```
Q1. 打开购物车，你的常态是？（HOARD / SUSCEPT）
  A) 塞了 30 多件，打算某天清空但永远没那天      → HOARD-H / SUSCEPT-H
  B) 5 件左右，最近想买的才会放                   → HOARD-M / SUSCEPT-M
  C) 基本是空的，要买就直接付款                   → HOARD-L / SUSCEPT-L
  D) 从来不看购物车，我都在直播间下单             → LIVESTREAM-H

Q2. 双 11 凌晨你通常在做什么？（FOMO / LIVESTREAM / SUSCEPT）
  A) 盯表跨零点，购物车已预演三遍                 → SUSCEPT-H / FOMO
  B) 刷直播间抢满减                                → LIVESTREAM-H
  C) 简单加两件想了很久的必需品                    → FRUGAL-H
  D) 睡觉，双 11 已经连续欺骗我七年了             → FRUGAL-H / SUSCEPT-L

Q3. 刚搬家，家里空荡荡，你的反应？（HOARD / SUSCEPT / SECONDHAND）
  A) 按笔记博主的清单一口气配齐氛围感             → SUSCEPT-H / HOARD-H
  B) 每周添一两样，慢慢有家的味道                   → HOARD-M
  C) 先去二手平台淘，能省一半                     → SECONDHAND-H / FRUGAL-H
  D) 就一张床一张桌子，够用就行                   → HOARD-L

Q4. 你刷到"最后三分钟上链接"的直播时，本能反应是？（LIVESTREAM / SUSCEPT / FRUGAL）
  A) 手已经在屏幕上了，反正可以退                   → LIVESTREAM-H / RETRN 隐含
  B) 看一眼价格，比外面便宜才下                   → LIVESTREAM-M / FRUGAL-H
  C) 听他 PUA 完我反而冷静                         → SUSCEPT-L
  D) 直接划走，我不吃这套                           → LIVESTREAM-L / SUSCEPT-L
```

> 完整 22 题在实施阶段填充，参考 FPI 题库结构（每维 3-4 题，每题 4 选项，每选项 1-3 维度加分）。

### 6.2 Gate 题（隐藏触发 + LIVESTREAM 维度兼用）

```
Q_gate. 上一次认真买点什么给自己是什么时候？
  A) 就是刚才／今天                             → HOARD-H / FLAUNT-M
  B) 这周／这个月                                 → HOARD-M
  C) 三个月前，想不起来具体什么                   → HOARD-L
  D) 我真的想不起来了                             → ZERO$ 触发候选（结合 FRUGAL 三题全低触发）
```

## 7. 分享海报设计

### 7.1 视觉方向

- **主色调**：深色底（墨绿 #0F1E17 或深灰黑 #111214）+ **金色币符**装饰元素（古典金 #C9A227），呼应"古法黄金"氛围。
- **币符元素**：右上角半透明叠加一个"¥""$""€""₿"混合旋转图形，做氛围，不做喧宾夺主。
- **雷达图**：6 维雷达图沿用 SBTI 视觉语言，但数据颜色改为金色渐变（鎏金效果）。
- **类型代号**：大号英文代号居中，下方小字中文别名。
- **底部刺痛短句**：加深色描边，做成"发票体"字体（模拟小票），但**不要真实品牌/价格**。

### 7.2 文案结构

```
MPI · MONEY PERSONA INDEX
{中文类型名}（例：直播间人质）
————————————————
6 维雷达图
————————————————
{一句刺痛}（例：321上链接就是你的生命节拍器）
————————————————
扫码看看你是怎么输掉钱的
```

### 7.3 反差视觉关键点

- 左上角小标签：`富养自己 / 穷养自己 / 互相养活`（根据 FRUGAL + FLAUNT 的组合自动生成标签）。
- 右下角"总账单"装饰：一个虚构的"本月败家系数"，例：`败家指数 87/100`（数值来自 6 维加权，仅做氛围，不记录用户真实金额）。

### 7.4 禁止项

- 不模拟任何真实支付 APP UI。
- 不显示任何真实货币金额/账单金额。
- 不使用任何真实品牌 logo / 主播头像 / 平台界面。

## 12. 与 FPI / GSTI / FSI 的关系

| 测试 | 主题 | 风险 | 与 MPI 的差异 |
|------|------|------|---------------|
| GSTI | 性转人格 | 中 | 身份反差；MPI 是行为自嘲，没有身份错位 |
| FPI | 朋友圈人设 | 低 | 表达场域；MPI 是消费场域，两者可做"人设+消费"组合梗 |
| FSI | 原生家庭 | 中 | 成长型创伤；MPI 是当下行为，不涉及情感沉重 |
| MPI | 消费行为 | 低 | **最轻量、最普世、最适合收尾的家族成员** |

### 12.1 可能的联动玩法

- **SBTI × MPI 联动结果页**：显示"作为一个 ZZZZ 死者，你在消费上是 LIVE! 直播间人质"——两代号连起来就是用户朋友圈文案。
- **FPI × MPI 对照**：`QSLIF 精神中产 × LUXUR 精致中产` 高度同源，可做"你的朋友圈和你的钱包供词一致"的二级彩蛋。
- **矩阵分享模板**：SBTI/GSTI/FPI/MPI 四个代号可拼成 4-code 身份卡（类似 MBTI 四字母），作为 SBTI 家族的终极识别符。

### 12.2 MPI 在矩阵中的战略位置

- **入门级**：新用户从 MPI 进入家族的门槛最低（话题安全、不需要性别选择、不需要家庭背景）。
- **留存级**：老用户测完 SBTI/GSTI 后，MPI 是最自然的"再测一个"选项。
- **长尾级**：消费话题天然有季节性（双 11 / 618 / 春节 / 发工资日），可重复唤起用户。
