# FSI — 原生家庭幸存者

> 目标：GSTI（性转）与 FPI（朋友圈）之后，第三枚爆款。切入当前小红书顶流情绪——"重养自己一遍 / 电子父母 / 断亲"——用测试的方式把这个讨论从"痛"翻成"懂"。
> 核心工程挑战：比起 FPI 的轻话题 / GSTI 的反差梗，FSI 触达的是**真实情绪痛点**。调性必须克制、兜底必须完备、禁用词必须明确。

---

## 1. 核心定位

### 1.1 命名

- **主推中文**：《原生家庭幸存者》
- **副标题**：你被养成了什么形状
- **备选中文**：《家庭出厂设置》《幸存者图鉴》《原生家庭毕业证》
- **英文代号**：**FSI**（Family Survivor Index）
- **路径建议**：`/new/fsi`
- **题量**：22 题
- **类型数**：18 个常规类型 + 1 个隐藏 + 1 个兜底
- **完成时长**：4-6 分钟
- **风险等级**：中（情绪沉重，调性需严控）

### 1.2 一句话解释

> 你不是性格古怪，你只是被你的家庭**出厂设置**过。测一测你被养成了什么形状，顺便知道自己要怎么**重养**回去。

### 1.3 用户动机

- **自我照镜**：想知道自己那些"奇怪点"有没有原生家庭的影子，把行为翻译成一个可命名的类型。
- **和解出口**：一个不需要看咨询师、不需要和家人吵架，就可以"把自己的家庭讲出来"的轻量容器。
- **抱团取暖**：结果页适合截图发在小红书的 #重养自己一遍 tag 下，评论区相互"这就是我"。
- **不装正能量**：相比 MBTI 式"你是 xx 领导力型"，FSI 允许用户承认"我就是个复印机娃 / 就是在逃孝子"。

### 1.4 调性边界（这一节是全文红线，反复引用）

**要做的（in-scope）**

- 用**自嘲幽默**讲"我被这样养大"，让用户笑中带刺。
- 用**行为模式**而非身份标签来描述类型（"你会下意识"优于"你是个 xx"）。
- 每个类型都要带**一句温柔出口**，指向"可以重养 / 可以断链 / 可以先笑一下"。
- 承认用户是**幸存者**（已经活过来了），不是"受害者"或"被毁掉的人"。

**不做的（out-of-scope）**

- 不指名攻击"你爸/你妈"，不把父母写成反派，不发"控诉信"型文案。
- 不用"原生家庭罪"、"毒母"、"爹味"、"凤凰男父母"等明确攻击性词汇。
- 不暗示"你的问题都怪你家"（这是创伤叙事，不是我们要做的）。
- 不渲染自残、抑郁、极端情节；结果页不出现"你永远好不了"之类的钉子话。
- 不替用户做断亲 / 不断亲 / 生不生孩子的决定。
- 不鼓励用户拿结果去"对号入座爸妈"——测的是用户自己，不是父母。

**一句总调性**：FSI 不是让你哭的，是让你**笑着承认**——"哦，原来我被训练成这个样子，那我接下来怎么办。"

## 2. 爆款潜力分析

### 2.1 master spec 的战略位置

从 `2026-04-17-viral-tests-masterplan.md` Section 5.2：

- 排第 3 阶段，衔接在 GSTI → FPI 之后；用 GSTI 流量余温引入情感深度。
- 爆款潜力 ★★★★，风险标注"中（情绪沉重，调性需管控）"。
- 预计周期 2-3 周，与 GSTI 相当。
- Master spec 直接给出 6 维骨架（控制 / 温暖 / 性别偏向 / 经济施压 / 过度关注 / 代际传承）和 9 个种子类型（COPYX / REBEL / SAINT / LEAVE / CURE! / SILNT / DADY+ / MAMY+ / PICKR）。本 spec 的工作是在这些骨架上做内容深化与风控兜底。

### 2.2 小红书情绪验证信号

| 信号 | 数据口径 | 对 FSI 的启发 |
|------|---------|--------------|
| #重养自己一遍 | 280w+ 笔记，持续增长 | 结果页必须有"重养"相关的温柔出口，而不是只丢诊断 |
| #电子父母 | 近 90 天增长 2099% | 承认"原生父母可能不是支持系统"，但不直接骂他们——用"电子父母"这种第三方词 |
| #断亲 | 持续稳定热度 | `LEAVE` 类型直接对应；但调性要做成"我选择自己" 而不是"我恨他们" |
| #原生家庭顶配 | 反向梗，小众但爆发 | 可以作为一个反差梗类型（`BRAG+` 幸运儿）——允许"好家庭"的人也测 |
| #小镇做题家 / #小镇贵妇 | 常年热词 | 经济施压维度可以呼应到"攀比式养育"和"做题家人生" |
| #课代表 / #讨好型人格 | MBTI 分支热议 | `SAINT` / `PLEASE` 类型正中该场域 |

### 2.3 为什么 FSI 能爆

- **情绪贴合度最高**：80/90/00 后集体议题，SBTI 没覆盖到的"家庭维度"。
- **截图欲天然强**：结果页是一张"我被养成的形状"，比朋友圈 FPI 更有个人叙事感。
- **评论区互动强**：家庭话题天然可"在下面评论你的类型 + 你家一句经典台词"。
- **碰瓷钩子明确**：SBTI 讲"你是什么物种"，FSI 讲"你**怎么变成**这个物种的"——前因后果的延展。

### 2.4 爆款风险的反面

- 可能引来媒体批评"利用家庭创伤做流量"——必须在首页放显眼的免责声明 + 心理支持链接。
- 评论区容易变成"控诉爹妈大赛"——文案设计上要引导"我在重养 / 我在和解 / 我在笑"，不是"我恨 xxx"。
- 少数真实创伤用户可能被触发——结果页必须有**深色兜底模块**指向专业帮助。

## 3. 维度模型（6 维）

为了复用现有 `TestConfig` / matching 架构，FSI 沿用 6 维，每维 3-4 题。维度命名保留 master spec 的方向但做了调性收敛——不用"控制 / 过度关注"这种心理学白话，改成更贴小红书黑话的词。

| # | 维度 | Code | A 极（高分） | B 极（低分） |
|---|------|------|-------------|-------------|
| D1 | 遥控器强度 | `CTRL` | 被安排派——从专业到相亲都轮不到你选 | 放养派——想干嘛干嘛，出事了再说 |
| D2 | 体感温度 | `WARM` | 热灶台——表面吵吵闹闹但底下是爱 | 冷藏柜——一切井井有条但没人抱抱 |
| D3 | 性别出厂设置 | `GNDR` | 重男/重女型——家里位次由性别决定 | 去性别化——至少表面一碗水端平 |
| D4 | 经济牵引力 | `MNEY` | 钱即爱——钱事和感情事混为一谈 | 钱是钱——钱就是钱，不用来施压 |
| D5 | 能见度光强 | `LITE` | 聚光灯下——你的一切都被看着评着 | 透明人——你说了半天没人接茬 |
| D6 | 代际回音 | `ECHO` | 复刻派——你下意识就是你爹妈那套 | 反叛派——打死不做我爸我妈 |

### 3.1 维度细读（SBTI 腔）

**D1 遥控器强度 `CTRL`**

- 高分：你从小就知道"我的人生有草稿"。专业是定好的，工作是推荐的，对象是介绍的。你擅长在别人的轨道里表现优秀，但你一个人做决定就会蓝屏。
- 低分：你家实行放养。你选什么他们都说随你，出了事也就一句"你自己的选择"。好处是你确实独立，坏处是你独立得有点太早了。

**D2 体感温度 `WARM`**

- 高分：你家吵得像菜市场，但过年大家会在一张桌子上。你的"爱"理解是嘈杂的、管事的、一大家子热闹的。安静的关系让你没安全感。
- 低分：你家像一间 AirBnB：干净、讲理、各自回房。你小时候以为这叫"文明"，长大才发现自己不会抱人也不会被抱。

**D3 性别出厂设置 `GNDR`**

- 高分：你在家里的位次早就被性别写好了。是男孩就继承压力，是女孩就负责懂事。你可能自己没意识到，但你花了很多年才允许自己"按自己的性别活"。
- 低分：你爸妈至少表面上没分男女。你做你的事他们做他们的。这当然是运气，但你看朋友们鸡飞狗跳时偶尔也会有一种"我是不是错过了什么"的错位感。

**D4 经济牵引力 `MNEY`**

- 高分：你家讲钱。给你的每一块钱都有潜台词——"我们养你不容易"、"你要考虑一下家里"、"你表哥买房你呢"。你和钱的关系因此复杂得像人民大会堂。
- 低分：你家不怎么在钱上拉扯。缺就缺，有就有，没人把工资条当武器。所以你对钱反而钝感，可能到了 30 岁还不会讨价还价。

**D5 能见度光强 `LITE`**

- 高分：你在家是**被主演**的。你的成绩、胖瘦、对象、工作、朋友圈，全家都有直播权。你擅长表演，只是表演太久会忘了自己本来什么样。
- 低分：你在家是**背景板**。你考了好成绩他们也"嗯"，你分手了他们也不问。你独立得很早，但你很晚才学会"主动要关心"。

**D6 代际回音 `ECHO`**

- 高分：你说话的口气、生气的方式、对小孩的耐心度——你看着看着就发现，**你就是你爸/你妈**。这不可怕，可怕的是你有时候会觉得"好像也还行"。
- 低分：你立过誓"我绝不变成他们那样"。你在感情里、在花钱上、在带孩子上都在刻意反向操作。问题是你反得太用力，有时候也累。

### 3.2 Pattern 表示法

每个类型用 6 位 pattern string，次序 `CTRL / WARM / GNDR / MNEY / LITE / ECHO`，取值 `H`（高）/ `M`（中）/ `L`（低）。

### 4.2 类型调性要求（每个类型的四段式）

每个类型的正式文案写作时必须包含以下四段，与 FPI / SBTI 对齐：

1. **镜像描述**（2-3 句）：用第二人称"你"，白描典型场景（"你过年回家……"、"你听到妈妈说……"）。
2. **天赋优势**（2-3 句）：辛辣包装的好话——你被这样养，确实也养出了几个点。不要虚美，要戳准。
3. **隐秘代价**（2-3 句）：扎心点破短处，但**只批行为模式，不批身份**；不骂父母，不骂用户。
4. **一句温柔出口**（1 句）：全文最后一句——SBTI 用"刺痛"，FSI 改用**刺痛 + 出口**的组合。例：
   - ❌ SBTI 腔："你永远爱不好一个人。"
   - ✅ FSI 腔："你不是不会爱，你只是还在学怎么放下遥控器。"

### 4.3 类型之间的 pattern 分布校准注意

- `COPYX` 和 `PICKR` pattern 几乎一致——实施时需要调一位拉开距离（例：PICKR 的 `LITE` 改为 `L`，表达"没人认可"）。
- `SAINT` / `SOS!` / `MAMY+` 在高 CTRL + 高 WARM 区扎堆——靠 MNEY / LITE / ECHO 三位做区分。
- `REBEL` vs `LEAVE` 差别：前者还在反着对线，后者已经不接电话了。

## 5. 隐藏类型 + 兜底

### 5.1 隐藏类型：`BOSSY` 家族 CEO

- **中文名**：家族 CEO（内部备选：家里的总司令）
- **触发条件**：需要同时满足——
  1. D1 `CTRL` 高分 + D5 `LITE` 高分 + D6 `ECHO` 高分
  2. 特殊题 `hidden_gate`：**"你现在和父母的关系更像哪种？"** 用户选择 C："我反过来管他们了"
- **内核一句**：你小时候被控制，长大后把控制权抢过来了。过年由你拍板，红包由你分配，爸妈也要看你脸色。
- **调性**：带点黑色幽默，承认"权力反转"本身是很多成年人面对原生家庭的真实状态；不煽情，不上价值。
- **一句温柔出口**：你不用赢，你已经赢了——该练习的是松手。

> 为什么要做隐藏：给真正"被严控大"的用户一个**反转式的幽默出口**，让他们不用停留在"受害者"叙事里。

### 5.2 兜底类型：`FAMX?` 家庭乱码人

- **中文名**：家庭乱码人
- **触发**：matching fallback——当用户答案过于均衡或互相矛盾，无法稳定匹配任一常规类型时命中。
- **内核一句**：你的家庭像一次多次合并冲突的 git 记录，你自己都没理清谁负责什么。
- **调性**：别做成"你测不出来 = 你有问题"，要做成"你复杂 = 你活得比测试更大"。
- **一句温柔出口**：也许你不需要一个类型，你需要时间。

## 6. 题目设计原则 + 样题

### 6.1 设计原则

- **场景化**，不抽象。问"上次你妈打电话你的第一反应是什么"，不问"你对母亲的感受是什么"。
- **选项不要暗示对错**，4 选项每一个都可能是真的。
- **避免真实创伤画面**：不写家暴、打骂、酗酒等具体情节——用代偿性场景（催婚、比较、金句、沉默）覆盖即可。
- **每题附一句兜底**："没有一个选项完美描述你？选最接近的那个，测试在乎的是倾向不是精准。"（仅在首 3 题下方短提示，避免过度说教。）
- **22 题**，每维度 3-4 题，覆盖场景见下。

### 6.2 场景清单（22 题可覆盖）

- 过年回家 / 家庭聚餐 / 亲戚问话
- 父母金句（"我们都是为你好" / "邻居家的孩子" / "你以后会明白"）
- 职业 / 感情被安排或被评价
- 电话铃响的第一反应
- 分享好事 / 分享挫折时家里的反应
- 父母生病 / 变老 / 变弱时你的第一念头
- 给家里寄钱 / 家里问你要钱 / 拒绝要求
- 在外地 vs 回家时的状态切换
- 看到别人家庭时的情绪反应
- 独自在家 / 搬出去住之后的变化

### 6.3 样题 4 条

```
Q1（维度：CTRL 控制强度）
你大三那年选职业方向时，家里的态度更像哪一种？
  A. 早就帮你选好了，不选就是不孝              （CTRL: H / ECHO: H）
  B. 嘴上说随你，但每次都拐到"那个专业才稳定" （CTRL: H / WARM: M）
  C. 问了一次你怎么想，听完就不管了            （CTRL: M / LITE: M）
  D. 完全没问，你选的时候都不知道他们关不关心  （CTRL: L / LITE: L / WARM: L）

Q2（维度：WARM 体感温度 + LITE 能见度）
你在外地工作，一个人生病躺床上。你告诉家里了吗？
  A. 告诉了，他们立刻打电话追问到底，像审犯人 （WARM: H / LITE: H / CTRL: H）
  B. 告诉了，他们说"多喝水"就没下文了         （WARM: L / LITE: L）
  C. 没告诉，告诉了也没用                     （WARM: L / LITE: L / ECHO: L）
  D. 没告诉，不想让他们担心                   （WARM: H / SAINT 倾向）

Q3（维度：MNEY 经济牵引力）
家里打电话，寒暄两句突然问你"一个月能攒多少？" 你的胸口：
  A. 立刻紧——知道接下来要聊什么            （MNEY: H / CTRL: H）
  B. 有点无奈但也没事，随便报一个数         （MNEY: M）
  C. 正常回答，这种问题我家常聊             （MNEY: M / WARM: H）
  D. 我家不太问钱的事，所以有点懵           （MNEY: L / WARM: M）

Q4（维度：ECHO 代际回音）
```

## 12. 与 FPI / GSTI / MPI 的关系

### 12.1 调性矩阵

| 测试 | 主题 | 调性强度 | 刺痛强度 | 出口 |
|------|-----|---------|---------|------|
| SBTI | 综合人格 | ★★★★ | 纯刺痛 | 无（读者自洽） |
| GSTI | 性别反串 | ★★★★ | 反差刺痛 | 无（玩笑式） |
| FPI | 朋友圈 | ★★★ | 中度刺痛 | 无 |
| **FSI** | **原生家庭** | **★★★** | **刺痛 + 温柔出口** | **必须有** |
| MPI | 消费 | ★★ | 自嘲为主 | 无 |

FSI 是家族里**第一个**把"温柔出口"明文写进规范的测试。原因就是话题敏感度——其他测试可以纯刺痛收尾，FSI 不行。

### 12.2 测试矩阵串联

三个测试上线后，形成**人格三联立**：
- SBTI / GSTI：你**对外**是谁
- FPI：你**在朋友圈**是谁
- FSI：你**怎么被养成**现在这个你

结果页互跳使用共用的"测试家族"组件，让用户刷完一个自然被导到下一个。

### 12.3 不做 MBTI / 心理学化

- 不用 MBTI 四字母做内核结构（那是人格分类，FSI 是行为模式家族）。
- 不引用 ACE 量表 / 依恋类型 / 原生家庭创伤等真实心理学量表——我们是**自省工具**不是**诊断工具**。
- 不做"康复建议 / 5 个步骤帮你走出 xx"的教学长文——太说教、太自负，也过度承诺了。
