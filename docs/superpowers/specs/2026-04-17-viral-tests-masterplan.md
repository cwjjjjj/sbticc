## 一、背景与痛点

### 项目现状
- 已有 6 个测试：SBTI（人格） / Cyber（网络行为） / Desire（欲望） / Love（恋爱） / Values（价值观） / Work（工作）
- 技术栈稳定：React + TS + Vite + Tailwind + Framer Motion，Upstash Redis 排行榜，Stripe/面包多 付费（目前关闭）
- 2026-04-09 晚 SBTI 在微博/朋友圈/小红书刷屏，24 小时挤崩服务器两次，#SBTI人格测试# 阅读量 1946 万+

### 用户反馈的痛点
1. **推广难，吸引不到用户** —— 即便 SBTI 爆过一次，长尾乏力
2. **除 SBTI 外的 5 个测试做得粗糙**：
   - 命名平庸（cyber: KBKING/COCOON/HERMIT 直译英文；values: CHUANG/NOREUR 拼音首字母）
   - 完全没有类型图片（5 个测试的 `typeImages.ts` 都是空 placeholder）
   - 缺乏传播抓手（分享出去没有"一定要点进去看"的欲望）
3. **没有持续出新爆款的节奏** —— 性格测试有"每年刷屏一次"的规律（2016 柏拉图→2023 MBTI→2026 SBTI），需要抢占下一个窗口

### 本次目标
- 设计并上线 1 个**具备爆款相**的新测试（GSTI），借 SBTI 余温引爆
- 沉淀一套可复用的**爆款测试生产方法论**
- 留出后续 3 个方向（FPI/FSI/MPI）的骨架，方便按周期迭代推出

## 二、战略洞察（决定一切的底层认知）

### SBTI 爆款 5 要素（来自媒体复盘）
1. **低参与门槛**：31 题 3-5 分钟，无须注册登录
2. **高保真分享海报**：毛玻璃效果 + 15 维雷达图，长按即保存，天然的社交货币
3. **反差感标签**：抛弃正能量/精英化表述，用"死者/吗喽/送钱者/屌丝/伪人/哦不人"这种荒诞自嘲标签
4. **社交货币属性**：整个朋友圈都在晒，不参与就被落下
5. **戏谑解构权威**：碰瓷 MBTI，给用户一个"终于不用被框住"的宣泄出口

### 新测试必须复刻的要素
- 题量 20-28，3-5 分钟可完成
- 结果页高保真海报（毛玻璃/雷达图/大标签），长按保存
- 类型命名：英文代号（4-6 字符，可用符号 `! ? + - ♂ ♀ 数字`）+ 一针见血的中文别名
- 类型描述：延续 SBTI 家族调性（辛辣/自嘲/刺痛），不要正能量鸡汤
- 有一个可复用 SBTI 粉丝的**碰瓷钩子**

### 2026 社交平台热度关键信号
- **活人感、抽象、自嘲、反差、情绪驱动** 是年度关键词
- 小红书热度话题：#重养自己一遍（290w+）、#电子父母（90 天增长 2099%）、#断亲、#万物皆可 MBTI（19 亿+）、#显眼包
- 性转/错位/反串 类梗长期稳定（B站/微博的女扮男装、性别反转剧、性别交换AI等）
- 性别对立话题（捞女/凤凰男/扶弟魔/妈宝）常年是流量密码，但直接做对立有封号风险

### 爆款测试窗口期
- SBTI 余温预计持续 4-8 周（媒体仍在发解读文章）
- 抢在"SBTI 热度衰减前"推出联名感强的新测试，可以直接继承原用户池

## 四、第一阶段：GSTI 完整设计

### 4.1 核心定位

- **名字**：
  - 主推中文：《性转人格测试》
  - 备选中文：《反串物种图鉴》《镜像人格测试》
  - 英文代号：**GSTI**（Gender Swap Type Index）
- **一句话解释**：男生测出的全是女性物种（捞女、娇妻、绿茶），女生测出的全是男性物种（凤凰男、妈宝男、普信男）的反差测试
- **用户动机**：
  - 想试的：好奇"换个性别我是什么"
  - 想骂的：反串标签扣在自己头上的冒犯感
  - 想转的：反差梗自带截图欲，"我一个老爷们居然测出了扶弟魔"
- **调性分寸**：**中度冲击**
  - ✅ 用：捞女 / 娇妻 / 绿茶 / 普信 / 舔狗 / 妈宝 / 扶妹魔（性转）/ 海后 / 凤凰女（性转）/ 娇夫 / 雄竞 / 傻白男
  - ❌ 不用：拳师 / PUA / 北京小灵（极端词） / 任何指向真实群体的辱骂性标签
  - ✅ 保留梗感、保留刺痛、保留自嘲
  - ❌ 不做赤裸性别对立，不贬低任一性别

### 4.2 机制设计

- **入口流程**：首页 → 选择性别（男 / 女 / 不透露）→ 开始答题 → 结果
- **性别 → 结果池映射**：
  - 男 → 女性物种池（20 类型）
  - 女 → 男性物种池（20 类型）
  - 不透露 → 双池混合，类型池 = 40 类型（命中两边任一，结果最不可预测）
- **题目跨性别通用**：每道题场景中性，两性都能代入；不引导性别特征
- **匹配算法**：与 SBTI 同架构（6 维 pattern 向量 + 向各池类型计算相似度）

### 4.3 维度模型（6 维）

| 维度 | 代号 | 两极 | 说明 |
|------|------|------|------|
| D1 | G/T | Giver（给予）↔ Taker（索取） | 在关系中你更像提供方还是收获方 |
| D2 | H/C | Hot（热烈）↔ Cold（冷淡） | 情感表达的温度 |
| D3 | T/B | Top（主导）↔ Bottom（被动） | 关系中的权力位置 |
| D4 | K/S | Calculus（算计）↔ Sincere（真诚） | 是否对利益做精打细算 |

### 4.6 类型描述撰写原则（参考 SBTI / cyber / values 已验证调性）

每个类型描述包含四段（沿用已有测试结构）：

1. **开场定位**（1-2 句）：一针见血点出核心画像
2. **天赋优势**（2-3 句）：辛辣包装的"好话"（带讽刺的）
3. **隐秘代价**（2-3 句）：扎心点破短处
4. **一句刺痛你的话**（1 句）：全文最尖的收尾

**性转版特殊原则**：
- 描述正文用第二人称"你"
- 文案里刻意保留性别标签反差（"你是一个男的，但是..." / "你作为一个姑娘..."）
- 用梗，不用脏话；用反差，不用贬低
- 每个类型必须让目标用户看完后**既冒犯又想截图**

### 4.7 题目设计

- **总数**：22 题
- **分布**：每维度 3-4 题
- **场景覆盖**：相亲 / 送礼 / 吵架冷战 / 社交聚会 / 性暗示 / 分手挽回 / 朋友圈发什么 / 异性示好 / 送伴手礼 / 为谁花钱
- **题目原则**：
  - 不分性别也能答（"如果有人追你，你..."）
  - 答案 3-4 选项，不能一眼看出"选 A 我会被归为 XX"
  - 题干尽量口语化，避免书面化

#### 新增文件
- `gsti.html`：入口 HTML（复制 cyber.html 调整 title/meta）
- `src/GstiApp.tsx`：顶层 App，复用 `TestConfigProvider` + `AppInner` 模式

#### 修改现有组件
- `src/utils/matching.ts`：增加 `poolFilter` 参数，根据用户选择的性别过滤类型池
- `src/utils/shareCard.ts`：GSTI 海报加性别反差视觉（左上角性别标签 + "SWAP" 标识）
- `src/hooks/useQuiz.ts`：持久化用户选择的性别到 state（传递给 `computeResult`）
- `index.html` 或主导航页：加入"性转版"入口卡片

#### 域名 / 路径
- 推荐复用现有多入口架构，新开 `sbti.jiligulu.xyz/gsti`（类似现在的 /cyber /work 等）
- 或独立域名：`gsti.jiligulu.xyz`（搭建起来更有仪式感，便于被媒体截图，但会损失 SBTI 主站导流效率）
- **决策倾向**：先用 `/gsti` 子路径最大化导流，流量稳定后再考虑独立域名

## 五、其他方向骨架（待后续展开）

### 5.1 FPI — 朋友圈人设诊断（第二阶段）

- **中文名**：朋友圈人设诊断 / 你在朋友圈是什么物种
- **英文代号**：**FPI**（Feed Persona Index）
- **维度**：4 大维度 × 3 细分 = 12 维
  - Production 生产（频率 / 精修度 / 原创度）
  - Identity 人设（真实度 / 一致性 / 温度）
  - Engagement 互动（点赞力 / 评论力 / 回复力）
  - Strategy 策略（可见性 / 保鲜期 / 屏蔽度）
- **类型 20 + 1**：
  - `FILTR` 滤镜代工厂 / `9PIC!` 九宫格暴君 / `EMO-R` 深夜 emo 发电机
  - `FLEXR` Offer 展柜 / `CKIN!` 打卡 KPI / `3DAYS` 三天可见教主
  - `SUBMR` 朋友圈潜水员 / `LIKER` 手滑点赞奴 / `GHOST` 已读不回
  - `COPYR` 情感转发器 / `SELLR` 人形广告位 / `BABY!` 娃娃直播员
  - `FURRY` 毛孩家长 / `MUKBG` 吃播显眼包 / `TRVL9` 国旗收集家
  - `BLOCK` 屏蔽狙击手 / `REDBK` 小红书代打 / `JUDGE` 三观广播台
  - `QSLIF` 精神中产 / `NPC-F` 朋友圈 NPC
  - 隐藏：`0POST` 朋友圈坟墓（彻底放弃朋友圈的人）
- **题目**：24 题
- **传播钩子**：朋友圈本身就是社交产品；"你的朋友圈是什么档次"自带比拼欲
- **风险**：低

### 5.2 FSI — 原生家庭幸存者（第三阶段）

- **中文名**：原生家庭幸存者 / 你被养成了什么形状
- **英文代号**：**FSI**（Family Survivor Index）
- **维度**：6 维（控制程度 / 温暖度 / 性别偏向 / 经济施压 / 过度关注 / 代际传承）
- **类型 15+**（草案）：
  - `COPYX` 复印机娃（全盘复刻父母）
  - `REBEL` 反叛者
  - `SAINT` 孝子贤孙（自我牺牲型）
  - `LEAVE` 断亲先锋
  - `CURE!` 重养自己
  - `SILNT` 假装没事人
  - `DADY+` 父爱缺失选手
  - `MAMY+` 母爱过载选手
  - `PICKR` 挑剔型（总觉得自己不够好）
  - 更多待设计
- **调性**：情感共鸣强但**调性必须克制**，不能消极化（避免深夜 emo 加重抑郁）
- **小红书验证信号**：#重养自己一遍（290w+）、#电子父母（90 天增长 2099%）、#断亲
- **风险**：中（情绪沉重，需温柔解构）

### 5.3 MPI — 消费人格图鉴（第四阶段）

- **中文名**：消费人格图鉴 / 你是怎么把钱输给世界的
- **英文代号**：**MPI**（Money Persona Index）
- **维度**：6 维（囤货欲 / 炫耀欲 / 省钱欲 / 易被种草度 / 二手/二奢偏好 / 直播沉迷度）

  - `GOLDN` 黄金囤户（"富养自己"派）
  - `SETUP` 消费降级打工人
  - `NUBOY` 无用好物收藏家
  - `LUXUR` 精致中产
