# FPI — 朋友圈人设诊断

> 目标：作为 GSTI 之后的低风险接续测试。FPI 不碰性别对立、不碰家庭创伤，直接切入"我在朋友圈到底像个什么人"这个高分享场景。

## 1. 当前社交趋势校验

### 1.1 结论

- FPI 比 FSI 更适合做第二阶段：话题轻、可自嘲、可晒结果，不容易把评论区带到沉重冲突。
- "朋友圈人设"天然适合病毒传播：测试对象就是用户的分享场域，结果页可以反向成为一条朋友圈内容。
- 内容方向要强调"活人感 vs 预制朋友圈"、"真实人格 vs 完美人设"、"情绪价值 vs 功能信息"，不要只做晒吃喝玩乐分类。
- 合规边界要避开恶意挑动对立、炫富羞辱、地域/性别/职业污名化和鼓励开盒式窥私。

## 2. 产品定位

- **中文名**：朋友圈人设诊断
- **副标题**：你在朋友圈是什么物种
- **英文代号**：FPI（Feed Persona Index）
- **一句话解释**：测出你的朋友圈到底是生活记录、情绪广播、精修展柜，还是一片安静坟场。
- **主打钩子**：你的朋友圈不是生活，是你的对外发行物。
- **路径建议**：`/new/fpi`
- **题量**：24 题
- **类型数**：20 个常规类型 + 1 个隐藏类型 + 1 个兜底类型
- **完成时长**：3-5 分钟
- **风险等级**：低

### 2.1 用户动机

- 想知道自己在别人眼里"发朋友圈像什么人"。
- 想拿结果去反向吐槽自己：我不是不发，我是在维持神秘感。
- 想让朋友互相对号入座：你就是那个九宫格暴君。
- 想用测试结果发一条"不那么正经但不尴尬"的朋友圈。

### 2.2 不做什么

- 不判断用户现实阶层、长相、收入、学历。
- 不鼓励用户截图审判某个具体好友。
- 不把"不发朋友圈"写成社交失败。
- 不制造性别、地域、职业、年龄对立。

## 3. 维度体系

为了复用当前 `TestConfig` / matching 架构，FPI 采用 6 维，每维 `L/M/H` 三档。24 题平均覆盖，每维 4 题。

| 维度 | Code | 高分含义 | 低分含义 |
|------|------|----------|----------|
| 发布密度 | `POST` | 高频更新，生活随手上架 | 极少发布，动态长期停尸 |
| 精修强度 | `POLI` | 图文精修、排版、滤镜、构图 | 随手发，粗糙但真实 |
| 人设意识 | `MASK` | 明确经营形象，知道自己要被怎样看见 | 不太经营，想到什么发什么 |
| 情绪电压 | `EMOT` | 情绪外放，朋友圈承担表达出口 | 情绪收纳，动态偏信息化 |
| 互动渴望 | `ECHO` | 在意点赞评论，擅长抛钩子 | 不太等反馈，发完就走 |
| 边界控制 | `GATE` | 分组、三天可见、屏蔽、权限管理熟练 | 边界松，谁看都差不多 |

### 3.1 隐藏触发

- `0POST` 朋友圈坟墓：通过 gate 题触发。
- Gate 题：`你上一次认真发朋友圈是什么时候？`
  - A. 这周
  - B. 这个月
  - C. 半年内
  - D. 我已经不记得朋友圈入口长什么样
- 选择 D 直接触发隐藏类型，但仍可继续答题用于维度展示。

### 4.2 隐藏与兜底

| Code | 中文名 | 触发/用途 | 一句话 |
|------|--------|-----------|--------|
| `0POST` | 朋友圈坟墓 | gate 题 D | 你的朋友圈没有人设，因为你已经把展厅关灯了。 |
| `FEED?` | 内容乱码人 | matching fallback | 你的动态像多平台同步失败：什么都有一点，什么都不像你。 |

### 4.3 类型调性要求

- 每个类型描述用四段式：镜像吐槽 / 天赋优势 / 隐秘代价 / 一句刺痛。
- "刺痛"只能刺行为模式，不刺身份属性。
- 类型名尽量短、易截图、易喊朋友对号入座。

## 6. 分享与增长设计

### 6.1 结果页卖点

- 主标题：`我的朋友圈物种：{typeName}`
- 副标题：`你不是在发动态，你是在发行自己。`
- 三个可截图模块：
  - 人设浓度：展示 6 维雷达图
  - 刺痛短句：一句话适合发朋友圈
  - 好友识别：`谁是你的同类 / 谁最看不懂你`

### 6.2 分享海报

- 文案结构：
  - `FPI · FEED PERSONA`
  - `{中文类型名}`
  - 6 维雷达图
  - `你的朋友圈不是生活，是你的对外发行物。`
- 视觉方向：
  - 像一张被压缩过的朋友圈截图，但不要直接仿微信 UI。
  - 卡片上可以有"仅三天可见"、"分组可见"、"已读不回"等标签贴纸。
  - 不使用头像/昵称模拟，避免隐私暗示。

### 6.3 冷启动话术

- `测完才发现，我的朋友圈不是生活，是样板间。`
- `你朋友圈是哪种物种？我是三天可见教主。`
- `别装了，你发朋友圈前也在想人设。`
- `这不是 MBTI，这是朋友圈照妖镜。`

## 7. 合规与内容安全

### 7.1 禁止项

- 不写"穷酸"、"低端"、"底层"等阶层羞辱。
- 不写"老登"、"小仙女"等容易引发群体对立的称呼。
- 不引导用户点名审判具体好友。
- 不把亲密关系、职场关系、家庭关系写成明确攻击对象。
- 不做真实微信 UI 高仿，避免平台商标/界面混淆。

### 7.2 推荐表达

- 用"运营感"替代"虚荣"。
- 用"观众席"替代"舔狗/看客"。
- 用"边界管理"替代"阴暗/心机"。
- 用"发行物"替代"人设诈骗"。

### 8.1 建议优先级

- P0：维度 + 24 题 + 20 类型四段式描述。
- P1：分享海报做"朋友圈发行物"视觉组件。
- P2：好友相性，主题可改成"谁最看懂你的朋友圈 / 谁会把你屏蔽"。
- P3：独立首发文案与小红书模板。
